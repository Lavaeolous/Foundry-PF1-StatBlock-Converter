PF1 STATBLOCK CONVERTER
This Tool converts Stat Blocks from the SRD or PDFs to JSON Files compatible with FoundryVTT
Currently only for the following Game Systems:
  • PF1, Pathfinder 1. Edition
Tested with the following Setup:
  • Core Version FoundryVTT: 0.6.2
  • System Version PF1: 0.561
